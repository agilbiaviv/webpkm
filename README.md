# Template Website Puskesmas (CodeIgniter 4 + Tailwind CSS)

Yoo! 👋 Ini adalah template dasar untuk pembuatan website Puskesmas menggunakan **CodeIgniter 4** dan **Tailwind CSS**.  
Sudah lengkap dengan struktur frontend & backend, sistem login admin, menu untuk posting berita, menu dinamis dari database dll.

> 📌 Bisa jadi starting point untuk pengembangan website puskesmas.

---

## 🔧 Apa Aja Isinya?

- ✅ Framework CodeIgniter 4 terbaru
- 🎨 Frontend pakai Tailwind CSS — clean & responsif
- 🔐 Admin panel, login, dan kontrol konten
- 📑 Menu dinamis langsung dari database
- 🗂 Konten modular: Beranda, Profil, Sambutan, Visi & Misi, Struktur Organisasi, Inovasi, Layanan Kesehatan
- 🧾 Migrations & Seeders siap pakai
- 📁 Upload file udah disiapin juga

---

## 🚀 Gimana Cara Pakainya?

Kalau kamu baru mau mulai atau mau fork repo ini, silakan cek file [`SETUP.md`](./SETUP.md) buat panduan lengkap langkah demi langkah.

Isinya mulai dari:
- Instalasi `composer` & `npm`
- Setting file `.env`
- Menjalankan migrasi & seeder database
- Build Tailwind CSS

---

## 📎 Beberapa File Penting

- `composer.json` / `composer.lock` → untuk library PHP
- `package.json` / `package-lock.json` → untuk Tailwind dan JS
- `tailwind.config.js` → konfigurasi Tailwind
